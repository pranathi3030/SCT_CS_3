# SkillCraft Cyber Internship

## Task 03: Password Strength Checker

## Description

A Java-based password strength assessment tool that checks the security level of a password based on different criteria.

## Features

- Checks password length
- Detects uppercase letters
- Detects lowercase letters
- Checks for numbers
- Checks for special characters
- Displays password strength level

## Technologies Used

- Java
- Regular Expressions
- Scanner Class
